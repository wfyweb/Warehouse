关于我们

禹州市星航钧瓷有限公司以弘扬传统民族文化为宗旨，坚持传统制作工艺，利用任氏祖传钧瓷技艺制作高品位艺术品，科技含量高，技术力量雄厚，现有国家级美术大师三人，省级工艺美术大师五人，省级工艺美术家二人，工程师二人，技师四人，技工七人。是集艺术传承、科研开发，生产高档钧瓷艺术品于一体的企业，有独具特色的浓郁文化艺术氛围。
